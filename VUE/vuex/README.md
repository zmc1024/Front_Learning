# vuex
  是vue的公共数据源，是响应式数据，
  可做各个组件间的传值

  mutations 和 actions 的区别：
    mutations:是同步代码，会当作源码去使用
    actions:是异步代码，会去到队列里

## Project setup
```
npm install
```
